# Profile Picture Setup Instructions

## Step 1: Add Your Profile Picture

1. Take the profile picture image you want to use
2. Rename it to `profile.jpg`
3. Copy it to the `public` folder in your project:
   ```
   MyProfile/devendra-codes-spark/public/profile.jpg
   ```

## Step 2: Run the Development Server

Open a terminal in the project directory and run:

```bash
npm run dev
```
